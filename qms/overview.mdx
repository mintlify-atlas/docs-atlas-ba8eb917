El Sistema de Gestión de Calidad (SGC) de CALAIRE-EA define el marco normativo y documental bajo el cual se diseña y opera el servicio de Ensayos de Aptitud (EA) para gases contaminantes criterio (CO, NOx, SO2, O3). Está estructurado conforme a las normas ISO 17043:2023 e ISO 13528:2017, tomando como base de competencia técnica la NTC ISO/IEC 17025:2017 que rige al laboratorio CALAIRE. Esta página presenta el alcance del SGC, los estándares aplicados, el estado de la documentación obligatoria y las auditorías internas pendientes.

## Designación institucional

El laboratorio CALAIRE, adscrito a la Universidad Nacional de Colombia, ostenta la designación de **Instituto Designado para Ozono** mediante la **Resolución 263/2023**. Esta condición habilita a CALAIRE para actuar como proveedor acreditado de ensayos de aptitud y como referente metrológico en la evaluación del desempeño de laboratorios de vigilancia de calidad del aire en Colombia.

## Normas aplicadas

Las tres normas que articulan el SGC de CALAIRE-EA cumplen roles complementarios: la ISO 17043 establece los requisitos del proveedor de EA, la ISO 13528 rige el tratamiento estadístico de resultados y la ISO 17025 fundamenta la competencia del laboratorio organizador y sirve de referencia para los requisitos mínimos exigidos a los participantes.

| Norma | Título abreviado | Rol en CALAIRE-EA |
|---|---|---|
| ISO/IEC 17043:2023 | Requisitos generales para EA | Diseño del servicio, requisitos del proveedor y estructura del programa |
| ISO 13528:2017 | Métodos estadísticos para EA | Algoritmos de valor asignado, puntuaciones z/z′/ζ/En en CALAIRE-APP |
| NTC ISO/IEC 17025:2017 | Competencia de laboratorios | Base del laboratorio CALAIRE; referencia para participantes |

Para una descripción detallada de cada norma y su interrelación, consulte [Normas ISO aplicadas](/qms/normas-iso).

## Documentación requerida (F-GCM-03)

El control documental del SGC registra el estado de cada protocolo e instructivo bajo el código de formato F-GCM-03. A continuación se muestra el estado actual de los documentos obligatorios.

| Documento | Tipo | Estado |
|---|---|---|
| Protocolo Técnico CO | Protocolo por gas | Borrador |
| Protocolo Técnico NOx | Protocolo por gas | Borrador |
| Protocolo Técnico SO2 | Protocolo por gas | Borrador |
| Protocolo Técnico O3 | Protocolo por gas | Borrador |
| Manual de Transporte de Equipos | Instructivo | Pendiente |
| Actualización Portafolio de Servicios | Documento comercial | Pendiente |

Para el detalle de contenido y alcance de cada protocolo técnico por gas, consulte [Protocolos técnicos por gas contaminante](/qms/protocolos-tecnicos).

## Auditorías internas

<Note>
La próxima auditoría interna del SGC de CALAIRE-EA no tiene fecha confirmada (TBD). La programación se definirá una vez que los protocolos técnicos superen la fase de borrador y se complete la documentación F-GCM-03.

Las auditorías internas verifican el cumplimiento del programa de EA frente a los requisitos de ISO 17043:2023 y el Plan de Gestión de Calidad del proyecto (artefacto 08_Plan_Gestion_Calidad.md). Los resultados alimentan el registro de incidentes (F-GCM-09) y las acciones correctivas del SGC.
